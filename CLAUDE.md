# Project conventions

## Language

- **All commit messages and GitHub releases (tag names, titles, and bodies) must be written in English**, even when the conversation is in another language. The user may be speaking Spanish, but the repository's public record stays in English.
- Chat replies to the user follow the user's language.
- The mod itself is bilingual (English + community Spanish translation); user-facing in-game text and README stay bilingual as already established.
